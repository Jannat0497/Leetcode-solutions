## 📂 Structure

Each problem is organized in its own folder using the LeetCode problem number and title.

```text
leetcode-solutions/
│
├── 1-two-sum/
├── 11-container-with-most-water/
├── 100-same-tree/
├── 101-symmetric-tree/
├── 102-binary-tree-level-order-traversal/
├── 118-pascals-triangle/
├── ...
│
└── README.md
```

Each folder contains the solution file along with a problem-specific README whenever applicable.

## 🧩 Topics Covered

Problems span a wide range of core DSA topics.

- **Arrays & Strings** — Traversal, searching, sorting, prefix sums
- **Two Pointers** — Pair problems, palindrome, merge arrays
- **Sliding Window** — Fixed and variable window techniques
- **Binary Search** — Classic search, rotated arrays, search space reduction
- **Linked Lists** — Traversal, reversal, cycle detection, merge lists
- **Stacks & Queues** — Parentheses, monotonic stack, BFS
- **Hashing** — Frequency maps, sets, lookup optimization
- **Trees** — DFS, BFS, traversals, BST problems
- **Recursion & Backtracking** — Subsets, combinations, permutations
- **Bit Manipulation** — XOR, bitmasking, bit counting
- **Greedy** — Interval scheduling, optimization *(growing)*
- **Dynamic Programming** — Memoization & Tabulation *(growing)*

## 🚀 How to Run

**Clone the repository**

```bash
git clone https://github.com/Jannat0497/Leetcode-solutions.git
cd leetcode-solutions
```

**Navigate to any problem folder**

```bash
cd 1-two-sum
```

**Compile**

```bash
javac Solution.java
```

**Run**

```bash
java Solution
```
